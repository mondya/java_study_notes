# Vim常用命令

## Normal模式

- `hjkl`：左下上右

- `G`：跳转到最后一行，类似于End键

- `gg`：跳到第一行，类似于home键

- `ctrl+u`：向上翻页

- `ctrl+d`：向下翻页

- `{lineno}gg`：跳转到低==lineno==行

- `zz/zt/zb`：光标行设置为屏幕居中/屏幕第一行/屏幕最后一行

## 进入Insert模式

Normal模式下通过特定方式进入Insert模式

- `i`：代表“insert”，当前光标之前开始输入
- `a`：代表“append”，当前光标之后开始输入
- `o`：下方插入新的一行，然后开始输入
- `s`：删除当前光标的字符，然后开始输入
- `I`：在本行的开头输入
- `A`：在本行的末尾输入
- `O`：上方插入新的一行，然后开始输入
- `S`：删除当前行，然后开始输入

## 进入Command模式

Normal模式下按`:`进入Command模式

- `:w`：保存当前文件
- `:q`：退出
- `:q!`：放弃当前修改，退出
- `:wq`：保存当前修改，退出
- `:h{command}`：显示关于命令的帮助

==Esc==回到normal模式

## 进入Visual模式

- normal模式下按`v`进入可视模式
- 进入可视模式后可以用Normal模式下的移动命令选择文本
- 可视模式下`x/y`：剪切/复制；回到Normal模式下`P`：粘贴
- Normal模式下按`V`进入行可视模式，一次选中一整行，在需要选中多行时方便

==Esc==回到Normal模式
